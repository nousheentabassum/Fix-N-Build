# Fix n Build

## Overview

Fix n Build is a full-stack home services platform that allows users to browse services, submit booking requests, and manage home maintenance tasks through a responsive web interface.

## Features

- User authentication
- Service booking and request management
- Responsive frontend
- RESTful backend APIs
- Contact and inquiry forms
- Modular frontend and backend architecture

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

## Getting Started

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

Open the frontend in a browser or run it using your preferred local server.
